Greeting RESTful Spring Service

REST Endpoints

GreetingResource

GET Paths

GET all greetings
GET all greeting objects as JSON or XML depending on request's Accept header.
Pass the request with Accept: application/json or Accept: application/xml
GET all greeting objects as JSON. Pass the request with Accept: application/json
GET all greeting objects as XML. Pass the request with Accept: application/xml
GET all languages
GET the greeting for given language such as spring/greetings/turkish/1
GET the greeting for given language such as spring/greetings/turkish/2
GET the greeting for given language using a query such as resources/greetings/query?language=turkish
GET greeting count 1
GET greeting count 2

POST Paths

POST a new greeting using language and greeting such as resources/greetings/arabic/assalamualaikum.
POST a new greeting using an entity

PUT Paths

PUT an existing greeting with new using language and greeting
